# 🎓 School Management System

A simple School Management System developed using Java AWT that allows users to manage student records through a graphical user interface. The application enables adding, viewing, and managing student information efficiently.

## 🚀 Features

- Add new student records
- Store student details (Name, Age, Grade)
- View all student records
- Clear input fields and records display
- User-friendly GUI using Java AWT

## 🛠️ Technologies Used

- Java
- AWT (Abstract Window Toolkit)
- Object-Oriented Programming (OOP)

## 📋 Project Overview

This project demonstrates the implementation of a basic student management system using Java AWT components such as:

- Frame
- Label
- TextField
- Button
- TextArea
- ActionListener

The system maintains student records in memory using an ArrayList.

## ⚙️ How to Run

### Prerequisites

- Java JDK 8 or above
- Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans) or Command Prompt

### Compile the Program

```bash
javac SchoolManagementSystemAWT.java
```

### Run the Program

```bash
java SchoolManagementSystemAWT
```

## 🖥️ Application Functions

### Add Student
Enter:
- Student Name
- Student Age
- Student Grade

Click **Add Student** to save the record.

### View Students
Click **View Students** to display all stored student records.

### Clear
Click **Clear** to reset all input fields and clear the display area.
